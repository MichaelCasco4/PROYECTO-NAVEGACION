📱 Simple Navigation App (Jetpack Compose)

📌 Description

This is a basic Android application developed using Kotlin and Jetpack Compose.
The main goal of the project is to demonstrate navigation between multiple screens using Navigation Compose

🎯 Features
3 functional screens:
Home Screen
Details Screen
Profile Screen
1.Navigation between screens
2.Modern UI using Material 3
3.Interactive components (buttons, icons, cards)

🧭 Navigation Flow
Home → Details → Profile
Details → Home
Profile → Home / Back
